# CAPSULA Investor Brief

## One-Liner

CAPSULA Studio is a commercial AI-native coding platform that turns software sessions into deployable capsules: apps, agents, web workers, mobile previews, WASM kernels, and GitHub release packages.

## The Problem

AI can generate code, but teams still struggle to turn generated code into real product artifacts. Most tools stop at files. Builders still need runtime setup, preview routes, manifests, issue triage, release evidence, and deploy handoff.

## The Solution

CAPSULA makes every build a capsule. A capsule carries source, runtime lane, preview surface, worker contract, AI tool metadata, WebAssembly plan, issue trail, and deployment destination.

## Why Now

AI coding has moved from novelty to workflow. The next bottleneck is not code generation; it is durable execution, preview, trust, and deployment. CAPSULA sits at that layer.

## Product

The current product includes:

- commercial React web studio
- Python runtime server
- local preview server
- MCP-style AI tool server

## Wedge

The first wedge is a local + GitHub-first CAPSULA Studio that helps users create and preview real app capsules quickly. The second wedge is hosted collaboration and persistent deployment. The third wedge is a capsule marketplace for apps, agents, and workers.

## Competitive Position

CAPSULA competes with AI coding apps, online IDEs, low-code builders, and deployment dashboards. Its difference is the capsule abstraction: code plus runtime plus preview plus manifest plus deploy evidence.
