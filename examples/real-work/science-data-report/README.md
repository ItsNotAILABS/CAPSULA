# Science/Data Report Real Work Example

This example proves CAPSULA can support analytical work, not only UI demos.

## Run

```bash
python examples/real-work/science-data-report/report.py
```

## Upgrade path

Install the professional stack when heavier work is needed:

```bash
python -m pip install -r requirements-platform.txt
```

Then extend the report with NumPy, Pandas, Polars, SciPy, SymPy, Plotly, notebooks, or FastAPI endpoints.
